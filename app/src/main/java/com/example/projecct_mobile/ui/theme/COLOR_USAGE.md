# Utilisation des couleurs dans l'application

## Couleurs définies dans Color.kt

- `DarkBlue` (0xFF1A237E) - Bleu foncé principal
- `LightBlue` (0xFFE3F2FD) - Bleu clair
- `GrayBorder` (0xFFE0E0E0) - Gris pour les bordures
- `LightGray` (0xFFF5F5F5) - Gris clair pour les fonds
- `White` (0xFFFFFFFF) - Blanc
- `Black` (0xFF000000) - Noir
- `AccentBlue` (0xFF2196F3) - Bleu d'accentuation
- `RedHeart` (0xFFE53935) - Rouge pour les favoris
- `Red` (0xFFFF0000) - Rouge pour les erreurs

## Vérification de cohérence

Toutes les couleurs sont accessibles via:
```kotlin
import com.example.projecct_mobile.ui.theme.*
```

Les couleurs sont utilisées directement dans les composables, ce qui est correct.
